# Frontend App (@ai-dev-agent/frontend)

This is the Next.js frontend application for the AI Dev Agent.

## Getting Started

1.  Navigate to the root of the monorepo.
2.  Install dependencies: `pnpm install`
3.  Run the development server: `pnpm --filter @ai-dev-agent/frontend dev`

The application will be available at [http://localhost:3000](http://localhost:3000).
