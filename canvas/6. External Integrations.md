6. External Integrations

6.1 LLM Providers

OpenRouter

Integration: HTTP REST calls with bearer tokens

Docs: OpenRouter API Reference

Use Cases: Multi-model fallback, cost/performance optimization

Notes: Monitor quota usage, handle 429 rate limits gracefully

LiteLLM

Integration: Local proxy service translating SDK calls to various vendor APIs

Repo: LiteLLM on GitHub

Use Cases: Offline-first dev, vendor switching without code changes

6.2 Search & Scraping

SearxNG

Integration: Query via /search?q=term&format=json

Repo: SearxNG GitHub

Use Cases: Metadata-rich search results for RAG ingestion

cURL

Usage: Direct HTML fetch for pages blocked from API scraping

Options: Proxy (-x), UA spoof (-A), cookie files

6.3 RAG & Indexing

Llama-Index

Integration: Python package (pip install llama-index)

Docs: Llama-Index Documentation

Use Cases: Document ingestion (Markdown, PDF, JSON); vector retrieval

LangChain

Integration: For text2SQL and orchestration (pip install langchain)

Docs: LangChain Docs

6.4 UI Ecosystem

Warp

Plugin API: Define custom commands via YAML manifest

Docs: Warp Plugins

VSCode/Cursor AI

Extension API: Webview panels, commands, context menus

Docs: VSCode Extension Docs

6.5 Database & Storage

SQLite

Integration: Built-in in Python (sqlite3) and Node.js (better-sqlite3)

Use Cases: Store metadata, session logs, index references

FAISS

Integration: Python (pip install faiss-cpu)

Usage: Efficient vector similarity search

6.6 Authentication & Security

OAuth 2.0 / API Keys

Secure LLM provider access; store credentials encrypted locally

Rotate keys periodically

Local Encryption

Optionally encrypt chat logs and DB with user-supplied passphrase
