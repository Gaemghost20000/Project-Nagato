10. Roadmap & Milestones

Phase 1: Foundation (Weeks 1-4)

Week 1:

Set up monorepo and basic service scaffolding (Chat Service, LLM Gateway).

Implement configuration management and environment variable support.

Create initial Docker Compose for local development.

Week 2:

Develop FileManager service with CRUD and watch mode.

Spin up SearxNG and implement basic Scraper Service endpoints.

Week 3:

Integrate Llama-Index for simple document ingestion and retrieval.

Build JSONL chat export functionality.

Week 4:

Implement Text2SQL Service with basic NL→SQL conversion.

Establish automated tests for core services.

Phase 2: UX & Integrations (Weeks 5-8)

Week 5:

Develop Warp Terminal Plugin commands (ai chat, ai file, ai scrape).

Create VSCode extension scaffold with sample Chat webview panel.

Week 6:

Wire up Chat Service to UI plugins; enable live streaming responses.

Enhance Scraper Service with curl fallback and proxy support.

Week 7:

Implement optional encryption for storage; build key management UI.

Add Git integration to FileManager.

Week 8:

Build RAG context injection into chat workflow; demo complete RAG query.

Conduct user testing and gather feedback.

Phase 3: Stability & Extensibility (Weeks 9-12)

Week 9:

Harden error handling, retries, and offline queueing for remote calls.

Add health-check endpoints and centralized logging.

Week 10:

Develop Plugin SDK and sample plugins (Formatter, Data Importer).

Implement plugin management UI.

Week 11:

Cross-platform packaging: build installers for macOS, Linux, Windows.

Set up CI/CD pipelines for automated builds and tests.

Week 12:

Documentation sprint: finalize README, API docs, and design spec.

Release v0.1.0 and onboard early adopters.

Future Phases (v0.2+)

Analytics & Telemetry (Optional Opt-In)

Advanced RAG: Support hybrid search (vector + keyword).

Mobile Companion App: Lightweight client for on-the-go access.

Cloud Sync: Securely sync encrypted data across devices.
